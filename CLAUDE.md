## Architecture

The app is a single-page calculator. All state lives in one hook; all computation is pure and stateless.

```
src/
  types/tax.ts              — All TypeScript types + TAX_RATES_2026 constants (source of truth for every rate)
  lib/tax-calculator.ts     — Pure calculation functions; no React dependency
  lib/utils.ts              — cn() (Tailwind merge), formatCurrency(), formatPercent()
  hooks/useTaxCalculation.ts — Single useState holding TaxInputs; result derived via useMemo
  components/
    RevenueForm.tsx          — Left column: CA input, activity selector, fiscal options (VL, ACRE, TVA, parts)
    PurchaseList.tsx         — Left column: CRUD list of professional purchases with per-item VAT rate
    TaxSummary.tsx           — Right column: result cards + threshold warning banners
    TaxBreakdown.tsx         — Right column: static explanation of rates, thresholds, coverage
    ui/                      — shadcn/ui primitives (Button, Card, Input, Label, Select, Switch, Badge, Separator, Tooltip)
  App.tsx                   — 2-column layout, wires hook → components
```

**Data flow:** `useTaxCalculation` → props → `RevenueForm` / `PurchaseList` (mutations) and `TaxSummary` / `TaxBreakdown` (display). No context or global store.

**Adding a new shadcn/ui component:** copy the pattern from an existing `src/components/ui/*.tsx` — they are hand-written Radix + CVA wrappers, not generated by the CLI.

## Tax rates (France 2026)

All rates are in `src/types/tax.ts → TAX_RATES_2026`. Update only this object when rates change.

| Catégorie | Cotisations | Versement libératoire | Abattement IR | Plafond CA |
|---|---|---|---|---|
| Vente marchandises (BIC) | 12,3 % | 1,0 % | 71 % | 203 100 € |
| Services (BIC) | 21,2 % | 1,7 % | 50 % | 83 600 € |
| Libéral BNC (régime général) | 25,6 % | 2,2 % | 34 % | 83 600 € |
| Libéral CIPAV | 23,2 % | 2,2 % | 34 % | 83 600 € |

**TVA franchise de base** (inchangée en 2026 après abandon de la réforme) :
- Services : seuil base 37 500 € / majoré 41 250 €
- Vente : seuil base 85 000 € / majoré 93 500 €

**ACRE** : −50 % sur cotisations sociales la 1ère année (taux porté à −75 % à partir du 1er juillet 2026).

**Barème IR progressif** (tranches dans `baremIR`) : 0 % / 11 % / 30 % / 41 % / 45 %.

**Impôt mode barème** : abattement forfaitaire appliqué sur le CA, résultat divisé par le nombre de parts (quotient familial), impôt calculé par tranche, multiplié par les parts.

**TVA mode assujetti** : TVA collectée = CA × 20 % (taux standard). TVA déductible = somme des TVA sur achats. TVA due = max(collectée − déductible, 0).

Sources officielles : [URSSAF](https://www.urssaf.fr), [impots.gouv.fr](https://www.impots.gouv.fr), [service-public.fr](https://entreprendre.service-public.gouv.fr).
